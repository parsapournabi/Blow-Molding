Parameters Hex Address: (Big Endian)

1- DI Read/Write 03/10H --> 40E-40F  DI1~DI14 (bit0=DI1 and bit13=DI14, 1 == enabled, 0 === disable)
2- DO Read 03 --> 03H --> 412-413 (bit0=DO1, bit4=DO5, 1 == enabled, 0 === disable)
3- Monitor Status 03H (Encode PUU (Scaled), Speed (rpm), Torque %, Average Load %, Peak Load %) --> 12-13, 14-15, 16-17, 18-19, 1A-1B
4- Alarms (referrer P0-01) 03H ---> 02-03 Displays Hex Alarm Code seven segment (for detail see P0-01)
5- Home First Speed 03/10H ---> 50A-50B 0~20000 (r/min but 2000.0) 32 bit decimals
6- Home Secondary Speed 03/10H ---> 50C-50D 0~20000 (r/min but 2000.0) 32 bit decimals
7- PATH#1 Data 03/10H --> 606-607 (32 signed bit decimal)
8- PATH#2 Data 03/10H --> 60A-60B (32 signed bit decimal)
...
9- PATH#63 Data 03/10H --> 737-738 (32 signed bit decimal)
10- Speed#0 03/10H --> 578-579 0~60000 0~60000 (r/min but 6000.0)  (16 bit decimal unsigned)
11- Speed#1 03/10H --> 57A-57B 0~60000 0~60000 (r/min but 6000.0)  (16 bit decimal unsigned)
...
12- Speed#15 03/10H --> 596-597 0~60000 0~60000 (r/min but 6000.0)  (16 bit decimal unsigned)
13- ACC/DEC#0 03/10H --> 528-529 1~65500 (millisecond 16 bit decimals)
14- ACC/DEC#1 03/10H --> 52A-52B 1~65500 (millisecond 16 bit decimals)
...
15- ACC/DEC#15 03/10H --> 546-547 1~65500 (millisecond 16 bit decimals)

16- P3-06 Enabling DI1~14 to software 03/10H --> 30C-30D (bit0=DI1 and bit13=DI14, 1 == software, 0 == hardware)  (default = 3B0F) NOTE: send it when Servo drive power on
17- JOG Speed 03/10H --> 40A-40B (0~5000 r/min) 20 means 20rpm
18- JOG ACC 03/10H --> 144-145 (1~65500) millisecond
19- JOG DEC 03/10H --> 146-147 (1~65500) millisecond

20- Torque Limit Register 03/10H --> 118-119 (-300~300 %) Torque percent
